# Data sources and redistribution

## New York City

The published table is derived from open MTA ridership and public route-stop sequence data. This repository includes the derived result table, but not the raw hourly source extract.

## Changsha / CZT

Station and edge tables are included because they are small and necessary for reproducibility. The nationwide WorldPop GeoTIFF is not included. The processed station-service population table and processing metadata are included.

## Licence note

The repository's MIT licence applies only to code. Original transport and population datasets retain their provider licences and attribution requirements. Derived result tables are published for research reuse with source attribution.
